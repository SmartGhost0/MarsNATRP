How To Use
-----

* Linux / Unix
    Recommand using with option "-m" ([DNS pointer mutation method])
    Run `sudo chinadns -m -c chnroute.txt` on your local machine. ChinaDNS creates a
    UDP DNS Server at `0.0.0.0:53`.
